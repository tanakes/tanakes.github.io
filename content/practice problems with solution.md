# ✅ SOLUTIONS

## Chapter 1 Solutions

**1.** x̄ = (12+18+15+22+19+14)/6 = 100/6 = **16.67**

**2.** Ordered: 3, 5, 7, 8, 9, 11, 13. n=7 (odd), median = 4th value = **8**

**3.** Frequency: 4→3 times, 7→3 times. **Modes: 4 and 7** (bimodal)

**4.** Data: 8, 10, 12, 14, 16 (ordered)
- a) Range = 16-8 = **8**
- b) x̄ = 12; Σ(x-x̄)² = 16+4+0+4+16 = 40; s² = 40/4 = **10**; s = √10 = **3.16**
- c) Q₁ position = (5+1)/4 = 1.5 → Q₁ = (8+10)/2 = 9; Q₃ position = 3(6)/4 = 4.5 → Q₃ = (14+16)/2 = 15; IQR = 15-9 = **6**

**5.** Midpoints: 15, 25, 35, 45, 55; n = 35
- a) x̄ = Σ(fm)/n = (75+200+420+315+165)/35 = 1175/35 = **33.57**
- b) Median class: n/2 = 17.5 → 3rd class (30-40); L=30, F=13, f=12, C=10
  Median = 30 + [(17.5-13)/12]×10 = 30 + 3.75 = **33.75**
- c) **Modal class: 30-40** (highest frequency = 12)
- d) Using shortcut: Σ(fm²) = 1125+5000+14700+14175+8250 = 43250
  s² = [43250 - (1175)²/35]/34 = [43250-39446.43]/34 = 111.93; s = **10.58**
